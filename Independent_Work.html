  • Robotic Arm - Control and Collision Avoidance [ code ]

    Robotic Arm Kinematics, Collision Avoidance | C++, ROS

    Tools: Robot operaing System, Smart actuators, Gazebo

    This project aims to control 4 Dof manipulator via joystick controller. This leverages smart actuator to control the individual joints, utilizing the forward Kinematics to publish the joint angles to get the position of the end-effector. Achieved the linear motion of the end‑effector for the plane and collision avoidance algorithm for the save operation.
